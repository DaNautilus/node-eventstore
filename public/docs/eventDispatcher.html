The eventdispatchers task is to publish undispatched events to the used publisher 
implementation. 

Example:



 dispatcher = new EventDispatcher(options);
 dispatcher.useLogger(logger)
           .usePublisher(publisher)
           .useStorage(storage)
           .start();
